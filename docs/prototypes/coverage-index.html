设计说明

  • 两视图共享同一套 OKLCH 配色与视觉规范，切换视图时体验连贯
  • 4 态颜色编码（达标绿✓ / 接近琥珀⚠ / 未达标红✗ / N/A灰—）双重表达，一眼定位覆盖盲区
  • 仪表盘含完整 AI 覆盖收敛面板（Gap 队列 + 并行调度 + 轮次 + Delta），树表格含浮动 AI 状态栏入口
  • AI 覆盖收敛：多 Gap 并行处理（上限 10），每个 Gap 独立仿真精确计算 Delta，闭环后通过 Diff Review 提升测试到正式目录
