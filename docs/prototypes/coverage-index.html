三套布局方案
A
树表格
可展开/折叠的层级模块树，右侧多列覆盖率指标。最接近 IMC 报告原貌，信息密度最高。
  • 8 列覆盖率指标 + 模块名列
  • 每格显示 percentage + (covered/total)
  • 颜色编码：达标/接近/未达标/N/A
  • 可折叠展开模块树
  • 最紧凑，工程师原生体验

B
旭日图 + 详情
旭日图展示覆盖率分布，点击模块后右侧显示详情面板。视觉层次最强。
  • 同心环 = 模块层级深度
  • 弧段颜色 = 覆盖率达标状态
  • 可切换 8 种指标查看
  • 点击模块显示详情面板
  • 最直观，适合演示和概览

C
仪表盘 + 下钻
概览卡片 + 趋势图 + 树表格 + 未覆盖项 + AI 建议面板。最全面的一站式布局。
  • 8 个概览卡片（含达标状态）
  • 覆盖率趋势折线图
  • 紧凑树表格
  • 未覆盖项列表（按指标切换）
  • AI 分析建议面板
