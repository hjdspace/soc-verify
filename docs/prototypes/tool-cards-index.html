三套设计方案
A
Terminal Native
等宽字体为主，终端风格输出块。密集、可扫描、工程师原生体验。
  • 全等宽字体，终端美学
  • 状态点 + 工具名 + 摘要行
  • 终端式输出块（bash/grep）
  • 色彩编码区分工具类型
  • 最紧凑，信息密度最高
查看原型 →
B
IDE Panels
结构化卡片，图标徽章 + 分区标签。类 VS Code 面板风格，视觉层次清晰。
  • 图标徽章 + 标题 + 状态药丸
  • 分区标签（Input/Output）
  • 圆角卡片 + hover 效果
  • Sans-serif 标题 + Mono 代码
  • 视觉层次最强，最易扫读
查看原型 →
C
Conversational Flow
极简风格，工具融入对话流。微妙的分类色调，渐进式展开。类 Claude/Cursor 风格。
  • 无边框折叠态，仅状态点 + 文字
  • 展开后才出现内容容器
  • 最低视觉噪音
  • 展开/折叠过渡平滑
  • 最优雅，最不干扰对话流
