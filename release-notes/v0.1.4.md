# Release Notes — v0.1.4

**发布日期**：2026-07-19  
**版本**：0.1.4  
**上一版本**：[v0.1.3](./v0.1.3.md)

---

## 概述

SoC Verify v0.1.4 聚焦于上下文选择、运行时模型切换与多 provider 凭据支持，并对 IPC RPC 路由与仿真管理进行全套扩展与重构。本版本新增文件/文件夹上下文选择、OptionDock 预设管理提示与预览、关闭行为持久化、用户消息技能 chip 展示等体验优化；同时统一替换硬编码颜色为主题变量、深化前端工作流模块，整体提升主题系统一致性与前端架构清晰度。

**提交统计**：9 次提交（自 v0.1.3 以来）

---

## 新功能

### 上下文选择

- 新增文件/文件夹上下文选择功能，便于在会话中快速引用项目资源

### 仿真选项与预设

- 优化 OptionDock 预设管理功能，新增提示与预览效果

### 应用体验

- 新增关闭行为持久化与托盘菜单重置选项

### IPC RPC 路由与仿真管理

- 新增全套 IPC RPC 路由
- 仿真管理模块重构与扩展

### 会话与消息

- 用户消息新增技能 chip 展示，可视化所应用的技能

### 运行时模型切换与凭据

- 新增运行时模型切换能力
- 支持多 provider 凭据管理

---

## 重构

- **ui**：优化 UI 样式与标题栏功能，新增测试用例
- **theme**：统一替换硬编码颜色为主题变量，重构主题系统
- **renderer**：深化前端工作流模块

---

## 已知限制

- 仅支持 Windows 平台打包（macOS / Linux 理论支持但未验证）
- omp 引擎以 git submodule 内嵌，不支持独立更换
- 单用户桌面应用，不提供 Web/移动端/多用户协作
- EDA 工具集成完全依赖插件，平台不内置任何 EDA 工具适配
