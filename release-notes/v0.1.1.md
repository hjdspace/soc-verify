# Release Notes — v0.1.1

**发布日期**：2026-07-13  
**版本**：0.1.1  
**上一版本**：[v0.1.0](./v0.1.0.md)

---

## 概述

SoC Verify v0.1.1 是 v0.1.0 后的首个迭代版本，聚焦于会话管理增强与 UI 体验优化。本版本新增了会话历史管理、工具卡片展示、源码控制工作流等功能，并对会话恢复与存储逻辑进行了深度重构。

**提交统计**：8 次提交（自 v0.1.0 以来）

---

## 新功能

### 会话历史管理
- 新增会话历史管理功能，支持恢复和删除历史会话
- 历史会话列表展示，一键恢复或删除

### 工具卡片与语法高亮
- 新增工具卡片功能，AI 工具调用结果以卡片形式展示
- 集成 highlight.js 语法高亮支持，代码块自动着色

### 会话模型持久化
- 会话模型选择持久化，记住用户上次选择的模型

### 源码控制工作流
- 新增源码控制工作流功能

---

## 重构

- **ai-session**：重构会话管理，支持恢复历史会话并优化 UI
- **session**：重构会话恢复与存储逻辑，优化体验
- **ipc,store**：重构会话 ID 匹配与持久化逻辑，实现懒加载运行时会话
- **RightPanel**：优化会话标签页加载指示器逻辑

## 已知限制

- 仅支持 Windows 平台打包（macOS / Linux 理论支持但未验证）
- omp 引擎以 git submodule 内嵌，不支持独立更换
- 单用户桌面应用，不提供 Web/移动端/多用户协作
- EDA 工具集成完全依赖插件，平台不内置任何 EDA 工具适配
