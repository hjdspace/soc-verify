## SoC Verify v0.1.8

本版本新增内置覆盖率解析器插件，增强覆盖率导入/合并的目录浏览体验，并修复 GitHub Release CI 发布流程。

## Highlights

- **内置覆盖率解析器**：新增 built-in coverage parser 插件，支持解析 IMC、VCS urg、vcover 等多种 EDA 工具的覆盖率报告，无需依赖外部插件即可完成覆盖率数据解析
- **覆盖率目录浏览**：新增目录浏览功能，支持覆盖率导入与合并时的目录选择，提升交互体验
- **技能内容读取**：新增 `readSkill` procedure 与 `readSkillContent` 方法，支持通过文件路径读取技能内容
- **设置面板增强**：SettingsPanel 新增内容加载与展开/折叠切换，优化技能查看体验

## Fixes

- 增强 CoverageManager 错误处理，报告生成失败时创建 meta.json 并优雅降级
- 修复覆盖率测试在无报告生成器场景下的处理，确保优雅降级
- 修复 GitHub Action Release 发布流程：上传资产前先创建 draft release，解决 `release not found` 错误

**Full Changelog**: https://github.com/hjdspace/soc-verify/compare/v0.1.7...v0.1.8
