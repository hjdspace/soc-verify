## SoC Verify v0.1.9

本版本新增源代码管理（SCM）操作能力与 MCP 服务器配置管理，并重构插件适配器架构以支持 MCP 集成。

## Highlights

- **源代码管理（SCM）**：新增文件暂存、取消暂存、丢弃和提交功能，提供完整的 Git 工作流操作，用户可直接在应用内完成版本控制操作
- **MCP 状态查询**：新增 `getMcpStatus` 命令与 API，支持查询 MCP 服务器运行状态，便于监控 MCP 集成健康度
- **MCP 配置管理**：新增 MCP 配置读写模块，支持项目级与用户级 MCP 服务器配置，灵活管理不同层级的 MCP 服务
- **覆盖率注册表**：新增覆盖率注册表，实现按项目缓存 CoverageManager 实例，优化多项目覆盖率管理性能
- **错误分析模块**：新增错误分析会话工厂与 prompt 模板，抽离错误分析逻辑，提升代码可维护性与复用性

## Refactor

- 重构插件适配器导入路径，将插件适配器从 `host/plugin-discovery` 迁移到 `plugin-adapters` 目录，统一导入路径
- 拆分 `router-context` 导出逻辑，将业务逻辑下沉到对应服务模块，降低 IPC 层耦合度
