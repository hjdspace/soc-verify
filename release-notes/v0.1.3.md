# Release Notes — v0.1.3

**发布日期**：2026-07-17  
**版本**：0.1.3  
**上一版本**：[v0.1.2](./v0.1.2.md)

---

## 概述

SoC Verify v0.1.3 聚焦于内置技能系统、多模态图片输入、AI 思考过程可视化与打包工程化完善。本版本引入内置技能/扩展目录机制，支持平台随附技能无需手动安装即可发现与加载；新增多模态图片处理能力，用户可通过拖拽将图片送入 LLM 会话；AI 助手的推理思考过程现在以独立区块展示；同时大幅完善了 native addon 自动下载、国内镜像加速、系统托盘等打包与工程化基础设施。

**提交统计**：59 次提交（自 v0.1.2 以来）

---

## 新功能

### 内置技能/扩展系统

- 新增内置扩展目录解析函数与目录支持
- 增强技能发现逻辑，支持内置技能自动发现与加载
- AgentOptions 与 InitConfig 新增 `additionalExtensionPaths` 配置
- SelectedSkill 新增 `builtin` 来源类型
- electron-builder 配置新增 `built-in-extension` 打包

### 多模态图片处理

- 新增 FileTree 图片拖拽支持，可将图片直接拖入会话
- 增强 handlePrompt 图片处理流程，支持图片送入 LLM
- 调整 CSP 策略以放行图片源

### AI 思考过程展示

- 新增 ThinkingBlock 组件，独立展示 assistant 推理过程
- 集成 ThinkingBlock 到 assistant 消息渲染
- 增强消息内容提取，将 thinking 内容分离处理

### 模型输入覆盖配置

- 新增 `buildModelInputOverrideConfig` 函数
- 实现模型输入覆盖配置能力

### 流式光标与性能优化

- 新增 memoized StreamingCursor 组件
- 新增光标闪烁动画（globals.css）
- 实现消息更新与持久化节流，优化大流量会话性能

### 项目文件管理

- 新增 Git 忽略文件处理，ProjectManager 集成 git ignore 检测
- FileTreeNode 新增 `gitIgnored` 属性标记

### 会话与日志

- 增强会话事件日志与摘要能力

### SOC 验证环境生成技能

- 新增 SOC 验证环境生成 SKILL.md
- 新增 `run_extract_mod_io.sh` 用于 mod_io 提取
- 新增 `env.json.template` 环境配置模板

### 应用体验

- 新增应用图标与系统托盘支持

## 已知限制

- 仅支持 Windows 平台打包（macOS / Linux 理论支持但未验证）
- omp 引擎以 git submodule 内嵌，不支持独立更换
- 单用户桌面应用，不提供 Web/移动端/多用户协作
- EDA 工具集成完全依赖插件，平台不内置任何 EDA 工具适配
