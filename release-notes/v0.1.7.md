## SoC Verify v0.1.7

本版本引入完整的 AI 驱动覆盖率分析能力：从层级树数据模型、覆盖率导入、可视化 UI 到 Coverage Closure 闭环循环与 Test Promotion，构建端到端的 AI 验证收敛工作流。

## Highlights

- **覆盖率数据模型**：实现层级树结构与端到端覆盖率导入，支持会话级覆盖率状态合并
- **覆盖率可视化 UI**：新增树形表与 Dashboard 仪表盘，直观展示覆盖率层级与命中情况
- **AI Host Tools 与 cov:// URI**：新增 AI Host Tools 与 `cov://` URI Scheme，使 AI Agent 可读取覆盖率数据
- **Coverage Closure 闭环**：实现 AI 编排器驱动的覆盖率收敛循环，集成 UI 实时展示闭环状态
- **覆盖率导出**：新增 HTML 报告与 JSON 结构化导出能力
- **Test Promotion 闭环**：实现测试用例晋升与闭环后整合，完成 AI 驱动验证全流程
- **终端项目工作目录**：命令执行支持项目工作目录上下文

## Fixes

- 修复子进程退出码处理，使 handleTerminalExit 正确反映退出状态
- 为 agent 进程实现错误监听器，捕获子进程异常

## Refactor

- 重构 runsim 重试工具改用 TerminalManager，统一终端调度入口
