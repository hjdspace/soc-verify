## SoC Verify v0.1.10

本版本重点增强 MCP 配置管理能力、覆盖率分析功能与用例搜索性能，并补齐 Linux 平台支持。

## Highlights

- **多作用域 MCP 配置管理**：新增多作用域 MCP 配置管理能力，支持 MCP 服务器工具查看与配置重载，并优化连接错误提示，提升 MCP 集成可观测性
- **默认系统提示词展示**：新增默认系统提示词展示能力，并提供终端降级兼容修复，确保在不支持新特性的环境下也能正常工作
- **覆盖率报告能力扩展**：新增 CSV、grade、bins 等覆盖率报告解析与命令支持，增强覆盖率报告生成器，支持更多 EDA 工具报告格式
- **覆盖率分析工具注册**：新增覆盖率分析工具注册机制，新增 TestContribution 类型与 EDA 命令更新，为 AI Agent 提供更丰富的覆盖率分析能力
- **覆盖率调试与导入日志**：增强 coverage store 覆盖率分析能力，新增调试信息与导入日志处理，CoveragePanel 新增 grade 与 uncovered 标签页，新增 getImportLog procedure 提供详细导入结果
- **倒排索引高效搜索**：实现倒排索引高效文本搜索，新增 CaseIndexManager 优化用例索引性能，搜索新增子系统过滤能力，大幅提升大规模用例库的搜索体验
- **Linux 平台支持**：实现 Linux 输入法与 D-Bus 会话总线设置，改进 D-Bus 检测与处理，补齐 Linux 平台运行环境支持

## Fixes

- 修复 Windows 平台技能删除 bug
- 添加 MCPManager 实例空值检查，避免空指针导致的崩溃

## Refactor

- 重构 `searchCases` 使用 caseIndexManager，文件树更新时自动失效索引，保证搜索结果一致性
